# NeverLand
first repo

## C++ Program

This repository contains a simple C++ Hello World program.

### Building

To build the program, run:
```bash
make
```

### Running

To run the program, execute:
```bash
make run
```

Or run the executable directly:
```bash
./neverland
```

### Cleaning

To clean up build artifacts:
```bash
make clean
```

### Requirements

- A C++ compiler (g++ or clang++)
- Make build system
